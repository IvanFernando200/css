Selectores de atributos:

Esto se esta aplicando a los links del footer la ultima seccion Mas informacion relevante

Corta referencia: que tenga el atributo en cuestion 
              .selectores-atributos a[href] {
              color: #529;
              }
Normal: que tenga el atributo y su valor completo 
              a[href="#index"] {
              text-decoration: none;
              font-size: 2em;
              }
Inclusion: (*) que tenga incluido cualquier parte del url determinado 
              .selectores-atributos a[href*="getbootstrap"] {
              text-decoration: none;
              }
Empieza con: (^) que comience con una url deterinada 
              .selectores-atributos a[href^="http://"]
              {
              font-family: cursive;
              }
Finaliza con: ($) que termine con la url determinada 
              .selectores-atributos a[href$=".co/#css-syntax"] {
              color: green;
              }
Incrustado con espacios a los costados: el comodin(~) funciona con un determinado valor que se encuentra en medio de dos palabras separada por espacios; Generalmente para clases. 
              .selectores-atributos a[class~="atributo_style"] {
              text-shadow:2px 1px 3px #900;
              }
Con: El comodin del ( | ), pipe aplica el estilo si contiene el texto en cuestion en una lista separada por espacios y que despues del texto en cuetion venga un guion medio - 
              .selectores-atributos a[class|="menu"] {
              font-weight: bolder;
              }

Modelo de caja


        Caja de Linea:
        - Ocupan el espacio necesario para mostrar su contenido.
        - No tienen dimensiones modificables ( alto, ancho).
        - Permiten otros elementos a su lado.
        - Padding y margin solo empujan a elementos adyacentes en horizontal, 
          NUNCA EN VERTICAL.
        Caja de Bloque:
        - Ocupa todo el ancho disponible, lo que genera saltos de linea.
        - Tienen dimensiones modificables (alto, ancho).
        - No permiten otros elementos a su lado (aunque especifique un ancho, 
          sigue ocupando todo el espacio disponible a lo ancho, generando saltos 
          de linea).

      padding & margin just (right-left)---- for inline(linea)

Posicionamiento

            Tipos de Posicionamiento:
            - static(default)
            - relative
            - absolute
            - fixed
            - sticky
            Para mover los elementos posicionados se activan las propiedades:
            - top(vertical - eje Y)
            - bottom(vertical - eje Y)
            - left(horizontal - eje X)
            - right(horizontal - eje X)
            - z-index(profundidad - eje Z)

            Las propiedades top y left van a tener prefernecias por sobre bottom y right respectivamente.
            
            Estas 5 propiedades n ofuncionan con el valor de static.
Static: default.
Relative: tiene espacio reservado en el DOM, inicializacion del lugar donde se encuentra.
Absolute: No tiene espacio reservado en el DOM, inicializacion del DOM o de su elemento posicionado padre que tenga posicionamiento.
Fixed:Al igula que relative y absolute. Con el unico detalle de que se quede estancado.
Sticky:Al igula que relative(+ fixed) es por seccion y se aplica solo con un, top, left, right, bottom; que nos sirve como punto de referencia.(por ello no mueven el elemento en si).
Ejm: top: 0px; ademas el espacion reservado en el DOM se matiene pero es desplasable.
Hijo directo del body o que su contenido padre tenga haltura o espacio definido.

                rem - basada en la anchurad el la "m" de la fuente del elemento raiz (html)
                ex - basada en la altura de la "x" de la fuente del elemento
                ch - basada en la anchura del "0" de la fuente del elemento
                % - al tama침o del contenedor
              vw, wh, vmin, vmax - al tama침o del viewport

                vh - alto del viewport van de 1 a 100
                vmax - entre vw y vh toma el que tenga mayor valor
